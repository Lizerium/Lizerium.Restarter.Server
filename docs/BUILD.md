# Build

```bash id="u4j0n2"
dotnet restore
dotnet build -c Release
```

---

# `appsettings.json`

- Create `appsettings.json` inside [app/Lizerium.Restarter.Server](app/Lizerium.Restarter.Server)

```json id="yl0b2v"
{
	"StatsFilePath": "full_path_to_stats.json",
	"ServerExecutablePath": "full_path_to_flserver.exe",
	"RestartIfNoPlayersAfterMinutes": 360,
	"CheckIntervalSeconds": 1,
	"ApiSecretKey": "secret_key",
	"ApiPort": 52349,
	"Language": "En"
}
```

---

# Configuration Parameters

- `StatsFilePath` — full path to `stats.json`

Example:

```text id="ntwuv8"
C:\Freelancer\EXE\stats.json
```

The `stats.json` file is generated by your server using **LizeriumFLHook** with the statistics plugin enabled.

Example test file:

[stats.json](app/TestData/stats.json)

---

- `ServerExecutablePath` — full path to the server executable `flserver.exe`

Example:

```text id="a0v4t4"
C:\Freelancer\EXE\flserver.exe
```

---

- `RestartIfNoPlayersAfterMinutes` — restart server after this number of minutes with no online players

Example:

```json id="a8c2z4"
360
```

---

- `CheckIntervalSeconds` — monitoring check interval in seconds

Example:

```json id="l7nyz8"
1
```

---

- `ApiSecretKey` — security key for external API access

Used in request header:

```text id="yzqf7x"
X-Api-Key
```

Example:

```text id="x8x6m1"
342adzscaw
```

---

- `ApiPort` — internal HTTP API port

Example:

```json id="a2j6rx"
52349
```

---

- `Language` — console UI and log language

Supported values:

```text id="q5mgnt"
En
Ru
```

---

# Important Note

> [!IMPORTANT]
> `stats.json` must be saved using `windows-1251` encoding.

This is required for correct Cyrillic text rendering in ship names, systems, and player names on Windows Server environments.

---

# Run

After configuration:

```bash id="ud4zpn"
Lizerium.Restarter.Server.exe
```

---

# What It Does

The application automatically:

- monitors `flserver.exe`
- restarts server if process crashes
- restarts server after long inactivity
- displays online players in console
- exposes statistics via HTTP API
- supports English and Russian languages
